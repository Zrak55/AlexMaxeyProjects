Pixel Paladin Readme
Alexander Maxey

This is a work in progress game.  It is a survival-platformer game in whicht he character controls the blue
cube (The Pixel Paladin) and must survive for as long as possible by avoiding or destroying the enemies that
spawn and move around the room.  It has a score and life system.  Not all features are yet working correctly

To work on: accurate life tracker, start screen, game over screen, sword collision box, controls

Controls: WASD or Arrow keys to move, space to use sword
